Observations Help

Search
Search for observations based on text in any field, including attribute values and tag names (note that all searches are case-insensitive):
fooMatch observations that contain 'foo'.
"foo bar"Match observations that contain the phrase 'foo bar'.
foo and barMatch observations that contain 'foo' and 'bar'.
foo or barMatch observations that contain 'foo' and observations that contain 'bar'.
not fooMatch observations that do not contain 'foo'.
foo and not barMatch observations that contain 'foo', but do not contain 'bar'.
Sort field
IDSort matched observations using the unique observation ID.
CreatedSort matched observations using the creation time of the observation.
ModifiedSort matched observations using the last modification time of the observation, if any.
Modified bySort matched observations by the username of the person who last modified the observation, if any.
TagsSort matched observations so that observations that share the same set of tags are grouped together.
Sort direction
AscendingSort matched observations in ascending order.
DescendingSort matched observations in descending order.
First, Prev, Random, Next, Last
Use the navigation buttons to move through the set of matched observations. You may also use the left and right keyboard shortcuts to select the previous and next observation, respectively.
Operations > View Observation
Opens the current observation in a separate widget, handy if you wish to compare two observations side-by-side.
Operations > Manage Attributes
Opens the Attribute Manager, where you can modify the current observation's attributes.
Operations > Manage Tags
Opens the Tag Manager, where you can modify the current observation's tags.
Operations > Export Observations
Exports the current set of matched observations to a zipfile containing each observation's content, plus a CSV manifest file. Note that the export process can be time-consuming, so the export zipfile is downloaded separately using the Deliveries Widget. You will receive a notification when the delivery is ready.
Operations > Delete Observation
Deletes the current observation and closes any related widgets.
Operations > Help
Opens this help.
